# Application Simulation Entretien

Web app statique pour les mises en situation Audioprothese.

Contenu :

- `index.html` : point d'entree
- `styles.css` : interface responsive
- `data.js` : 8 cas patients + plan de vente
- `app.js` : navigation et logique d'interaction

Usage :

1. Ouvrir `index.html` dans un navigateur.
2. Entrer le prenom de l'apprenant.
3. Choisir `Patient` ou `Audioprothesiste`.

Ce que fait la v1 :

- role patient : fiche complete pour incarner un cas
- role audioprothesiste : coach pas a pas base sur le plan de vente
- debrief simple de fin de simulation

Ce que la v1 ne fait pas encore :

- synchronisation entre deux appareils
- scoring formateur
- export PDF ou DOCX

- `Deliverables/Plan de vente Audition-Conseil-66.pdf`
